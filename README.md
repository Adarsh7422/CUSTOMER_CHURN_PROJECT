# CUSTOMER_CHURN_PROJECT
Business Problem
Understanding why customers leave helps businesses improve retention and reduce revenue loss.

Tools Used
•	Python
•	Pandas, NumPy
•	Matplotlib, Seaborn
•	Jupyter Notebook

 Analysis Performed
•	Data cleaning and validation
•	Churn distribution analysis
•	Monthly charges vs churn comparison
•	Gender and contract-wise churn analysis

Key Insights
•	Higher monthly charges lead to higher churn.
•	Month-to-month contract customers churn more.
•	Long-term customers show better retention.

Conclusion
This analysis helps businesses understand customer behaviour and identify churn-driving factors. The insights can be used to design better pricing strategies, improve customer engagement, and reduce churn.
